About

WorkLog

Broadcast freelance tracker

Local-first app for ChromeOS. Track jobs, fees, meals and receipt totals for broadcast freelance work.

Your data is stored on this device. Use Backup to export your data regularly.

Using WorkLog on ChromeOS

Open WorkLog from its GitHub Pages address. To use it like an app, open the Chrome menu and choose Install WorkLog or Create shortcut, then enable Open as window.

No Linux Terminal is needed. Your jobs remain stored locally in this Chrome profile, so export a JSON backup regularly.
